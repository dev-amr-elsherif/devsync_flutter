## 1. üìñ Project Overview

**DevSync** is an innovative cross-platform mobile application developed using **Flutter**. It aims to revolutionize how Project Managers (PMs) and Developers connect by utilizing an **AI-driven matching ecosystem**.

The platform serves as a specialized MVP that focuses on **intelligent team formation**. Unlike traditional project management tools, DevSync prioritizes verifiable technical competence by analyzing real-world public GitHub activity, ensuring that project requirements meet the most qualified talent.

## 2. ‚ö†Ô∏è Problem Statement

The current recruitment and team-building landscape faces three critical challenges:

- **Verification Gap:** PMs struggle to evaluate the true technical proficiency of developers beyond self-declared CVs.
- **Discovery Friction:** Talented developers often go unnoticed if they lack "keyword-optimized" profiles, even if their actual code is superior.
- **Efficiency Loss:** Manual searching and vetting lead to wasted time, mismatched expectations, and project delays.

---

## 3. üí° Proposed Solution

DevSync addresses these issues by creating a **data-driven bridge**:

- **AI Chatbot Integration:** Simplifies project definition for PMs through natural language processing.
- **GitHub Evidence:** Uses the GitHub REST API to analyze public commits, languages used, and contribution patterns.
- **Smart Matching:** An AI algorithm that ranks developers based on their actual coding history relative to project needs.
